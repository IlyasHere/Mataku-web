# OcuSense

OcuSense adalah web prediksi risiko Dry Eye Disease berdasarkan pola tidur, tingkat stres, dan penggunaan perangkat digital menggunakan metode Fuzzy Mamdani dan Fuzzy Sugeno.

## Tujuan Project

Project ini dibuat untuk memenuhi Tugas Besar DKA dengan implementasi fuzzy logic secara manual atau from scratch tanpa menggunakan library fuzzy.

## Metode

- Fuzzy Mamdani
- Fuzzy Sugeno
- Fuzzifikasi
- Inferensi
- Defuzzifikasi
- Evaluasi hasil Mamdani vs Sugeno

## Dataset

Dataset yang digunakan adalah Dry Eye Dataset dalam format CSV.
